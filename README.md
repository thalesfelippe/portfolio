# Thales Felippe Portfolio

Static professional portfolio for Thales Felippe, Full Stack Developer.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS

## Requirements

- Static and responsive portfolio
- Bilingual interface: Portuguese and English
- Light and dark theme
- Ready for GitHub Pages with custom domain `thalesdevflp.com`

## Available Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
npm run lint
```
